# Voice Reader

Un'applicazione web per la sintesi vocale (Text-to-Speech) che converte il testo in parlato usando l'API Web Speech API del browser.

## Caratteristiche

- 🎤 **Sintesi vocale** - Converte testo in voce usando le voci del browser
- 🎙️ **Selezione voce** - Scegli tra le voci disponibili nel browser
- ⚡ **Controlli velocità/tono/volume** - Personalizza la lettura
- ▶️ **Controlli Play/Pausa/Stop** - Controllo completo della riproduzione
- 📱 **Responsive** - Funziona su desktop e mobile

## Tecnologie

- HTML5
- CSS3 (con variabili CSS e flexbox/grid)
- JavaScript (Web Speech API)

## Come usare

1. Apri `index.html` in un browser moderno (Chrome, Firefox, Safari, Edge)
2. Incolla o scrivi il testo nell'area di testo
3. Scegli la voce, velocità, tono e volume
4. Clicca "▶ Leggi" per iniziare la lettura

## Browser Supportati

- Chrome 33+
- Firefox 49+
- Safari 14+
- Edge 14+
